# Standards

**Purpose:** Technical standards for development, agents, and operations.

## The 3 Pillars

1.  **[DEVELOPMENT.md](DEVELOPMENT.md)** (The "How")
    *   **Audience:** Developers writing code.
    *   **Content:** Coding standards, Testing, Git, Linting, Data Validation.

2.  **[AGENTS.md](AGENTS.md)** (The "What")
    *   **Audience:** Agent Architects and Builders.
    *   **Content:** Architecture, Lifecycle, Memory, Coordination, Tools, MCP.

3.  **[OPERATIONS.md](OPERATIONS.md)** (The "Run")
    *   **Audience:** DevOps, SRE, Security.
    *   **Content:** Observability, Traceability, Evaluation, Security.
